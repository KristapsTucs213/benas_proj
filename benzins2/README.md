# Degvielas_proj

## Project setup

```
1. Noklonē projektu
2. npm install
3. cd backend
4. Uzstādīt attiecošo datubāžu info kexfile.js failā
5. npx knex migrate:latest - runno migracijas
6. node server.js - uzpildes stacijas cenu info funkcijas
7. node account_thing.js - account funkcijas
8. cd..
9. npm run serve

```
